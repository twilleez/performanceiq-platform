# Performance IQ — Elite Brand Identity

## Overview

Performance IQ is a premium athletic performance platform designed for elite athletes, coaches, and organizations. The brand identity reflects luxury, precision, and elite-tier performance standards.

### Font Family

**Primary Font:** Inter (Google Fonts)
- Clean, modern, highly legible

- Available in weights: 300, 400, 500, 600, 700

- Clear distinction for technical content

## Accessibility

- **Color Contrast:** All text meets WCAG AA standards (4.5:1 minimum)
- **Focus States:** Visible gold border on all interactive elements
- **Font Size:** Minimum 12px for body text
- **Line Height:** 1.6 for readability
- **Motion:** Respects `prefers-reduced-motion` preference

## Usage Guidelines

### Do's ✅
- Use gold for primary actions and success states
- Use emerald for health, readiness, and positive indicators
- Maintain consistent spacing using the spacing system
- Use Inter font for all UI text
- Apply shadows for depth and hierarchy
- Use the color palette consistently across all platforms

### Don'ts ❌
- Don't mix multiple accent colors in a single component
- Don't use text smaller than 12px
- Don't apply shadows to text
- Don't use colors outside the defined palette
- Don't apply filters or effects to the logo
- Don't stretch or distort the logo
